# Custom Neovim Configuration

## Keymaps

```<leader>ca```    : code action from lsp 
```<leader>ff```    : find files
```<leader>fg```    : live grep
```<leader>wg```    : save all and exit all
```<leader>d```     : lsp diagnostics

```[d```            : next diagnostics
```]d```            : prev diagnostics
